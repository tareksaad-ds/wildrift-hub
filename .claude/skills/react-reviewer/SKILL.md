# React Component Review Skill

When creating or editing any React component, always:

## Check these things
1. Component has proper TypeScript types on all props
2. No direct DOM manipulation (use refs properly)
3. useEffect has correct dependency arrays
4. Images have alt attributes
5. Buttons have descriptive labels
6. No console.log left in the code

## After any edit report
- ✅ What looks good
- ⚠️ What needs attention
- 🚫 Any critical issues

Keep the report short — max 5 bullet points.
